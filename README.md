# 👋 Fala, eu sou o Ithalo

Sou Product Manager na área de tecnologia educacional, trabalhando na interseção entre produto, estratégia e impacto real na vida das pessoas.

Gosto de construir produtos que não só funcionam bem — mas que fazem sentido no contexto de quem usa.

---

## 🧠 Como eu penso produto

Acredito que bons produtos nascem de três coisas bem equilibradas:

* Clareza de problema (antes de sair construindo solução)
* Proximidade com o usuário (de verdade, não só em discurso)
* Decisões conscientes de trade-off

No dia a dia, isso se traduz em:

* Descoberta contínua (discovery não é fase, é hábito)
* Escopo incremental bem definido
* Forte parceria com design e engenharia
* Uso de IA como alavanca real de produtividade 

---

## 🚀 O que eu tenho construído

Hoje trabalho com produtos digitais educacionais, especialmente em:

* LMS para disciplinas suplementares  (socioemocional, tecnologica, financeira, etc.)
* Experiências gamificadas de aprendizagem
* Ferramentas para gestão escolar 

Também venho explorando bastante o uso de IA para:

* Geração de especificações de produto
* Aceleração de discovery e refinamento
* Construção de dashboard de forma autonôma
* Padronização de processos
